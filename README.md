# Movies App

## Technologies Used
- Kotlin
- Android Jetpack (ViewModel, StateFlow, Room, Paging 3)
- Coroutines & Flow
- Hilt (Dependency Injection)
- Retrofit (Networking)
- Room for caching, RemoteMediator for paging
